# NexusDocs

La exportación JSON completa del workflow NexusDocs se incorpora como anexo técnico y/o archivo digital complementario. Este artefacto documenta la configuración real de la solución implementada en n8n y permite verificar su estructura, sus nodos, sus parámetros de operación y sus mecanismos de integración con Jira, Bitbucket, Confluence e inteligencia artificial.
